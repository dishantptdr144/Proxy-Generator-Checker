## ✨ Features

* ⚡ Fetch proxies from multiple public sources
* 🌐 Supports **HTTP**, **HTTPS**, **SOCKS4**, and **SOCKS5**
* 🚀 High-speed asynchronous proxy checking
* 📍 Country detection
* ⏱️ Latency measurement
* 🔥 Automatically separates fast proxies
* 📂 Automatic `results` folder creation
* 🎨 Clean colored terminal output
* 🧹 Removes duplicate proxies
* 💾 Saves working proxies by protocol
